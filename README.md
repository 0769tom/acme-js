# Acme javascript library

## Install

```
npm install acme-js
```

## Import

CommonJS

```
var AcmeJS = require('acme-js');
```

Front end

```
<script type="text/javascript" src="/browserify-acme.js"></script>

# 或者
<script type="text/javascript" src="/browserify-acme-min.js"></script>

# 访问
console.log(window.AcmeJS)
```
